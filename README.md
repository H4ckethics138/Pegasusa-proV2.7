## 🚀 **Quick Start**

</div>

### **Step 1: Installation** first of all you will have to clone this report
```bash
git clone https://github.com/H4ckethics138/Pegasusa-proV2.7.git
```

### **Step 2:  now you will have to change the directory**
```bash
cd Pegasusa-proV2.7
```

### **Step 3: Launch & Develop**
```bash
Python Main.py
```
